Intelligently Artificial F.C. - A Robocup Simulation League Team

16 Feb 2016, 11:26
Development
Java / ATAN / robocup / AI
http://github.com/ragnarula/robocup

This project was mine and James Frost’s coursework for Introduction to Artificial Intelligence. We were tasked with creating an AI football team using the ATAN interface to the Robocup 2D Soccer Simulation Server. Our goal was to come up with some sort of architecture that would allow us to seperate our design in to a collection of small simple components, each design to do just one clearly defined task. These “AI Components” are arranged sequentially, and once they finish processing a task, they hand the processed data over to the next component in the chain, being careful to make a clean hand-off and not keep and references.
